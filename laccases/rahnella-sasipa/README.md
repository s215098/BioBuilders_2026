# Investigating Laccases from Sasipas paper (the Hamburg paper) as potential enzymes for breakdown of BADGE

From this paper: https://doi.org/10.3389/fbioe.2025.1670548, and this dissertation: https://ediss.sub.uni-hamburg.de/bitstream/ediss/12070/1/Dissertation_Wongwattanarat_2025.pdf  we know that the laccase ItL-03 from Rahnella sp. (probably, it's metagenomic data from the beetle Ips typographus) can degrade 3 different substrates present in the epoxy resin (amine-crosslinked) RTM-6, which is used in aircraft industry. These substrates are NNBT, BBCM, and BMAP, and I put the files in data. The enzyme CueO from E. coli also showed promising results. 
All the enzymatic sequences are in the data folder.
